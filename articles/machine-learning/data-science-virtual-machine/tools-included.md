# <a name="what-tools-are-included-on-the-azure-data-science-virtual-machine"></a>Azure Data Science Virtual Machine 中包含哪些工具？

Data Science Virtual Machine 是在云中浏览数据和进行机器学习的最简单方法。 Data Science Virtual Machine 已预先配置了完整的操作系统、安全修补程序、驱动程序和常用的数据科学和开发软件。 可以选择硬件环境，选择范围包括低成本的以 CPU 为中心的计算机，以及具有多个 GPU、NVMe 存储和大量内存的极强大计算机。 对于具有 GPU 的计算机，我们安装了所有驱动程序，对所有机器学习框架进行了版本匹配以确保 GPU 兼容性，并且在支持 GPU 的所有应用程序软件中启用了加速。

Data Science Virtual Machine 预装有非常实用的数据科学工具：

## <a name="how-to-choosecondaenvironment"></a>如何选择 conda 环境  

> [!NOTE]
> 从旧版 Data Science Virtual Machine 切换到新版时，存在中断性变更。 

**从 Windows Server 2016 切换到 Windows Server 2019** </br> 

* conda 环境设置为 python37_default。 我们不支持 python2。 如果在 python conda 环境中使用 tensorflow 或 pytorch，请分别使用 py37_tensorflow 或 py37_pytorch。 如果仅使用 AzureML，请使用 azureml_py36_automl </br>

* 如果在 AzureML conda 环境中使用 tensorflow 或 pytorch，请分别使用 azureml_py36_tensorflow 或 azureml_py36_pytorch。 </br>

**从 Ubuntu 16.04 切换到 Ubuntu 18.04** </br>

* 如果在 azureml_py36 conda 环境中使用 tensorflow 或 pytorch，请分别使用 azureml_py36_tensorflow 或 azureml_py36_pytorch。 </br>

* 如果在 py37 conda 环境中使用 tensorflow 或 pytorch，请分别使用 py37_tensorflow 或 py37_pytorch。 </br>

* 否则请使用 python37_default；或者，如果仅使用 azureml_py36，请使用 azureml_py36_automl </br>
